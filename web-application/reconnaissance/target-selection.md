# Target Selection

***

### Exposure

Seberapa terbuka? Apakah bisa diakses pakai browser biasa? Apakah bisa tanpa header khusus? Apakah bisa tanpa login?&#x20;

### User Input Density

Seberapa banyak user input, bisa dari Input Form, URL, Parameter, JS Event, etc.

### Dynamic Behavior

Semakin dinamis maka semakin tinggi kemungkinan. Seperti apakah ada XHR, API Call, etc.

### Bussines Impact

Kalau rusak, efeknya apa? Seberapa parah? Data pengguna? Pembayaran? Atau hanya informational?

### Environment Signal

Prioritas pertama adalah Production, lalu Tanpa Environment, ang terakhir Staging

### Protection Signal

Apabila tidak ada protection misalkan dari WAF, itu prioritas utama.
